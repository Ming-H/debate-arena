# DebateArena - AI辩论竞技场

## 项目概述

DebateArena 是一个基于 Claude 的多智能体辩论系统，通过让多个 AI 辩手就争议性话题进行多轮辩论，帮助用户深入理解复杂问题的不同视角。

## 核心目标

1. **多立场辩论**：创建支持不同立场的 AI 辩手
2. **结构化流程**：主持人引导的有序辩论流程
3. **智能评判**：基于论点质量的客观评价系统
4. **可扩展性**：易于添加新话题和新辩手人设

## 核心组件

### 1. Debater（辩手）
- 角色：持特定立场的辩论参与者
- 职责：
  - 根据立场生成论点
  - 回应对方论点
  - 引用证据支持观点

### 2. Moderator（主持人）
- 角色：辩论流程控制者
- 职责：
  - 宣布辩题
  - 控制发言顺序
  - 提醒规则
  - 总结每轮要点

### 3. Judge（评判系统）
- 角色：论点质量评估者
- 职责：
  - 分析论点强度
  - 检查逻辑谬误
  - 评估证据质量
  - 给出综合判断

### 4. Arena（竞技场）
- 角色：系统协调者
- 职责：
  - 初始化辩论配置
  - 协调各组件交互
  - 记录辩论过程
  - 输出最终报告

## 辩论流程

```
1. 初始化阶段
   ├── 定义辩题
   ├── 创建辩手（分配立场）
   └── 设置轮次限制

2. 开场阶段
   ├── 主持人宣布辩题
   └── 各方阐述立论

3. 辩论阶段（多轮）
   ├── 正方发言
   ├── 反方回应
   ├── 主持人总结
   └── 重复N轮

4. 结辩阶段
   └── 各方总结陈词

5. 评判阶段
   ├── 分析论点质量
   ├── 评估逻辑一致性
   └── 生成综合结论

6. 输出报告
   ├── 完整辩论记录
   ├── 最佳论点
   └── 评判结论
```

## 评估维度

| 维度 | 描述 | 评分标准 |
|------|------|----------|
| 论点强度 | 核心观点的清晰度和说服力 | 1-10分 |
| 证据支持 | 事实、数据、案例的引用 | 1-10分 |
| 逻辑性 | 推理过程的严密性 | 1-10分 |
| 反驳质量 | 对对方论点的针对性回应 | 1-10分 |
| 表达清晰度 | 论述的条理和可读性 | 1-10分 |

## 使用示例

```python
from debate_arena import Arena

arena = Arena(
    topic="人工智能是否会对人类构成威胁？",
    rounds=3
)

result = arena.run()
print(result.report)
```

## 技术栈

- **语言**: Python 3.11+
- **AI模型**: Claude (Anthropic API)
- **配置管理**: pyproject.toml
- **依赖管理**: pip

## 开发原则

1. **模块化设计**：每个组件职责单一，易于扩展
2. **提示词工程**：精心设计提示词以获得高质量输出
3. **可配置性**：支持自定义辩手人设、评判标准等
4. **可观测性**：详细记录辩论过程，便于调试和分析
